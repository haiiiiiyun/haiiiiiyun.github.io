# 环境设置

TypeScript 是 JavaScript ES6 的超集，它添加了类型。

Angular 2(简称 ng2) 虽然提供了 ES5 (普通 JavaScript) API，但由于其本身是用 TypeScript 编写的，因此 ng2 的应用最好也应该用 TypeScript 编写。
